How it works
  1. Copy the URL of an event from TicketSwap or Ticketmaster
  2. Paste the URL in the appropriate form above
  3. Click Import to automatically extract event details
  4. The event will be added to your tracker with all available information
Supported Information
  • Event name and type
  • Venue and location
  • Date and time
  • Price and currency
  • Original ticket URL
  • Source attribution
